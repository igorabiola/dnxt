registerCallback (callback, frequency, count, ...)
Registra função para ser chamada no cliente em um tempo determinado

Parameters:

  • callback: (function) função a ser chamada
  • frequency: frequencia de chamada da função em segundos.
  • count: numero de vezes a callback deve ser chamada, zero para infinito.
  • ...: parametros a serem pasados para a função.

registerSensorCallback (robotID, port, frequency, callback, lowspeed)
registra nos sevidores o cliente para receber o valor de um sensor.

Parameters:

  • robotID: id do robo.
  • port: porta do sensor.
  • frequency: frequencia de chamada da função em segundos.
  • callback: nome da função a ser chamada para retorno do valor.
  • lowspeed: ( boolean ) indica se o sensor é de lowspeed.
